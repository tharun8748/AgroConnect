# 🌾 AgroConnect

AgroConnect is a full-stack MERN (MongoDB, Express, React, Node.js) web application designed to connect farmers directly with buyers and agricultural experts. The platform empowers farmers to list their crops, seek expert advice, and access real-time weather updates — all in one place.

---

## 🚀 Features

- 🧑‍🌾 **Farmer Profile Creation**
- 📦 **Product Listing (Crops, Fertilizers, etc.)**
- 💬 **Chat with Agricultural Experts**
- 📈 **Weather-Based Suggestions**
- 🛒 **Direct Buyer-Farmer Connection**
- 🔐 **Secure Authentication & Role Management**
